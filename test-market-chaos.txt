# MARKET MODULE CHAOS PLAYTEST
# Testing: shopping, prices, demonstratives, comparing items, buying by weight

# ============================================================================
# GOAL 1: Explore the market (market_explore)
# ============================================================================

# Basic exploration - go to a stand
voy al puesto de frutas

# ============================================================================
# GOAL 2: Ask about prices (market_ask_price)
# ============================================================================

# Ask price - singular item
cuanto cuesta la sandia?

# Ask price - plural items
cuanto cuestan las manzanas?

# Alternate phrasing for prices
a cuanto estan las fresas?

# ============================================================================
# GOAL 3: Use demonstratives (market_demonstratives)
# ============================================================================

# Near speaker - "this/these" (este/esta/estos/estas)
quiero estas manzanas

# Near listener - "that/those" (ese/esa/esos/esas)
dame esas naranjas

# Far away - "that over there" (aquel/aquella/aquellos/aquellas)
prefiero aquellas fresas de alla

# Wrong gender demonstrative (should still work if understandable)
quiero este naranja

# Demonstrative with pointing context
este platano se ve muy fresco

# ============================================================================
# GOAL 4: Compare products (market_compare)
# ============================================================================

# Go to vegetable stand for more comparison practice
voy al puesto de verduras

# Ask which are fresher
cuales tomates son mas frescos?

# Direct comparison question
estas papas son mas grandes que esas?

# Ask for recommendation
cual es mejor?

# ============================================================================
# GOAL 5: Make a purchase (market_purchase)
# ============================================================================

# Buy by weight - kilo
quiero un kilo de tomates, por favor

# Buy by weight - half kilo
me da medio kilo de cebollas

# Buy by quantity
quiero dos lechugas

# Buy specific amount - grams
cien gramos de ajo, por favor

# ============================================================================
# GOAL 6: Pay at checkout (market_checkout_pay)
# ============================================================================

# Go to checkout
voy a la caja

# Ask total
cuanto es el total?

# Pay with cash
pago con efectivo

# Alternative: pay with card
pago con tarjeta

# ============================================================================
# EDGE CASES - Chaos Testing
# ============================================================================

# Misspelling - should be understood
quero un kilo de papas

# Missing accent - common learner error
cuanto cuesta?

# Grammar error - noun-adjective agreement
las manzanas rojas son muy buena

# Informal request
dame un poco de fresas

# Overly formal
quisiera adquirir doscientos gramos de jamon, si es tan amable

# Compound command - move and ask
voy a la carniceria y pregunto el precio del pollo

# Compound command - greet and ask
hola, cuanto cuesta la carne de res?

# Using ticket system at meat counter (chaos test)
tomo un numero

# Asking vendor what's on sale
que tiene de oferta hoy?

# Asking for a bag
necesito una bolsa, por favor

# Saying goodbye
gracias, hasta luego

# ============================================================================
# UX TESTS - NPC Visibility and Clarity
# ============================================================================

# Go back to fruit stand to test NPC interaction
voy al puesto de frutas

# Talk to vendor directly
hablo con Dona Maria

# Ask about ripeness
estos platanos estan maduros?

# Ask for help choosing
cual me recomienda?

# ============================================================================
# ADDITIONAL DEMONSTRATIVE TESTS
# ============================================================================

# Neuter demonstratives
que es eso?

# Comparing with demonstratives
este o ese?

# Pointing at specific items
no esas, estas de aqui

# Mixed demonstrative phrases
estas naranjas son mejores que aquellas

# ============================================================================
# QUANTITY AND WEIGHT EDGE CASES
# ============================================================================

# Large quantity
quiero cinco kilos de papas

# Fraction
quiero un cuarto de kilo

# By piece vs weight confusion (watermelon is by piece)
cuanto cuesta el kilo de sandia?

# Non-standard quantity
quiero tres naranjas

# ============================================================================
# FINAL CHECKOUT AND RECEIPT
# ============================================================================

# Return to checkout for final payment
voy a la caja

# Complete purchase
pago todo

# Ask for receipt
me da el recibo, por favor

# Done
gracias
